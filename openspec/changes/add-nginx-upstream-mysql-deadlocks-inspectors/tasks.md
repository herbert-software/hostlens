## 2. mysql.deadlocks manifest（builtin/mysql/）

- [x] 2.1 新增 `src/hostlens/inspectors/builtin/mysql/deadlocks.yaml`（`mysql.deadlocks`）：`mysql --batch -N --connect-timeout=5` 跑 `SHOW ENGINE INNODB STATUS`，collector 内 parse「LATEST DETECTED DEADLOCK」段首行时间戳，**在采样时刻于目标机内**算 `deadlock_age_seconds`，冻结进输出（wave-2b 确定性录制硬条款：禁回吐需回放时按 `now()` 重聚合的时间戳明细）。**时间戳解析（design D2 修订）**：collector 用 GNU `date -d` 解析 INNODB STATUS 段头的 **ISO 形** `YYYY-MM-DD HH:MM:SS`（现代 MySQL 5.7/8.0 默认）；**不**声称兼容紧凑 `YYMMDD` 形（GNU date -d 无法把 `240102` 当 YY-MM-DD，是 overclaim）——非 ISO 形解析失败走 fail-loud（见下），目标版本格式由真机 Demo Path 验证。**输出键拍板（B1，消除「省略 vs -1」二义）**：恒发两键 `deadlock_detected`（bool）+ `deadlock_age_seconds`（int），`output_schema.required: [deadlock_detected, deadlock_age_seconds]` 两键都进；**无死锁时 collector END 分支恒发哨兵 `deadlock_age_seconds: -1`**（与 mysql.slow_queries 无慢查询恒发 `slow_query_count: 0` 同款），**禁止省略键**（省略会令 runner `output_schema` 校验失败 → exception，与「无死锁→ok」矛盾）。secret 声明 `HOSTLENS_MYSQL_PWD`，collector 内 `export MYSQL_PWD="${HOSTLENS_MYSQL_PWD:-}"`（**从不内联进 argv**），与 mysql.slow_queries 逐字对齐；参数 `host`（pattern `^[a-zA-Z0-9._-]+$`，default 127.0.0.1）/ `port`（default 3306）/ `user`（required，pattern）/ `lookback_seconds`（default 3600）——**无 `warn_count`**（finding 阈值是 `deadlock_age_seconds <= lookback_seconds`，不需 warn_count，与 proposal manifest 示例一致）；`parameters` 必须 `type: object` 包裹。**collector awk 段布局（B1）**：「LATEST DETECTED DEADLOCK」标记夹在 `------` 分隔线间、时间戳是标记后**第一条 ISO 日期行**（紧邻下一行是闭合分隔线），故 awk 必须 flag-on-marker + 匹配 `^[0-9]{4}-` 跳分隔线取首条 ISO 行（**禁单次 getline**——会取到分隔线致 date 解析炸、真死锁恒 exception）。fail-loud：mysql error / 非数字 age / 时间戳 parse 失败 → exit 1 + 空 stdout → status=exception（不静默报健康）；无死锁段 → `deadlock_detected=false` + `deadlock_age_seconds=-1` → ok。findings：`deadlock_detected == True and deadlock_age_seconds <= lookback_seconds` → warning（`deadlock_detected==False` 时 `and` 短路，-1 哨兵不会误触）。**密钥脱敏**：输出仅 bool/int 标量，不回吐 SQL/死锁现场文本；验收含「`HOSTLENS_MYSQL_PWD` 不进 argv / 日志 / 报告」断言（crosscheck `forbidden_flags` 已覆盖 ` -p`）。验收：`hostlens inspectors show mysql.deadlocks` 出 manifest + 参数表。
