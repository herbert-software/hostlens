  - **bool 必须出 JSON `true`/`false` 字面量**(`@@global.slow_query_log`/`@@global.log_output` 出 `1`/`0`/`ON`/`TABLE` 文本,collector 须在 shell 内映射成 `true`/`false`——直接出 `1`/`0` 会被 `{type: boolean}` 校验拒→健康实例误 exception)

  - `case` 数值校验 `slow_query_count`,非数值/空→`exit 1`+空 stdout→`exception`
  - manifest `description`/运行文档声明 `slow_query_log=ON` + `log_output` 含 `TABLE` 前提、SSH 上需远端 `AcceptEnv HOSTLENS_*`
